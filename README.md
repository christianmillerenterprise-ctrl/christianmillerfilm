# christianmillerfilm.com

The portfolio site for Christian Miller — a Next.js (App Router) project
styled with Tailwind CSS. Warm cream paper, editorial serifs, film grain.

## What's inside

- **`app/`** — Next.js App Router pages:
  - `page.jsx` — landing page with hero, manifesto, and scroll-through work
  - `work/[slug]/page.jsx` — per-shoot detail page (hero + description + gallery)
  - `inquire/page.jsx` — inquiry form (Formspree)
  - `not-found.jsx` — 404
  - `globals.css` — base styles, grain overlay, color tokens
- **`components/`** — `Nav`, `Footer`, `GrainOverlay`, `ShootCard`
- **`data/`**
  - `shoots.js` — **edit here** to change shoot titles, subtitles, descriptions
  - `photos.json` — auto-generated by the image processor (don't hand-edit)
- **`public/images/<slug>/`** — web-optimized photos (1800px wide, 80% JPEG)

## Quick start

```bash
# 1. Install dependencies
npm install

# 2. Run the dev server
npm run dev
# → open http://localhost:3000
```

## Deploying to Vercel (free)

1. Push this folder to a GitHub repo (private is fine).
2. Go to [vercel.com](https://vercel.com) → **New Project** → import the repo.
3. Framework preset: **Next.js** (auto-detected). No env vars needed.
4. Click **Deploy**. You'll get a `*.vercel.app` URL immediately.

### Pointing christianmillerfilm.com at it

1. Buy the domain (Namecheap, Cloudflare Registrar, or Google Domains — Cloudflare is cheapest, ~$10/yr with no markup).
2. In Vercel → **Project → Settings → Domains** → add `christianmillerfilm.com` and `www.christianmillerfilm.com`.
3. Vercel will show you DNS records to add. Paste them into your registrar's DNS panel.
4. SSL provisions automatically in ~30 seconds.

**Total recurring cost: ~$12/year** (just the domain). Vercel's hobby tier is free for personal portfolio use.

## Wiring up the inquiry form

The contact form posts to [Formspree](https://formspree.io), which is free for up to 50 submissions/month — more than enough for a photography inquiry form.

1. Sign up at formspree.io with your email.
2. Create a new form — name it "Christian Miller Inquiries" or similar.
3. Copy the form ID (looks like `xbjnpdqz` or similar).
4. Open `app/inquire/page.jsx` and replace the placeholder:

```js
const FORM_ID = "YOUR_FORMSPREE_ID"; // ← replace with your real ID
```

5. Redeploy (Vercel auto-deploys on `git push`).

Submissions now arrive at `christianmillerenterprise@gmail.com` (or whichever email you used to sign up at Formspree).

**Alternatives to Formspree** — if you ever want to switch: Basin, Tally, or Resend. All have similar free tiers.

## Editing content

### Changing shoot titles, subtitles, descriptions

Open **`data/shoots.js`**. Each shoot has:

```js
{
  slug: "bryan-and-amy",        // URL segment and image folder name
  title: "Bryan & Amy",          // display title
  subtitle: "A Pregnancy — …",   // small-caps line
  heroIndex: 0,                  // which photo is the hero (0 = first)
  description: [                 // paragraphs (one string per paragraph)
    "…",
    "…",
  ],
  photos: photos["bryan-and-amy"], // don't edit — auto-loaded from photos.json
}
```

Placeholder titles and descriptions were written to match the aesthetic and reading of the photos. Swap in your own voice — you're the author.

### Swapping or adding photos

Photos are auto-loaded from `/public/images/<slug>/`. To re-generate the curated set with different selections, edit and re-run the processor script:

```bash
python3 process_photos.py
```

(You'll find this script in the outputs folder that was shared alongside this site; it uses Pillow to resize from your raw folders. Re-running it overwrites the images and updates `data/photos.json`.)

### Changing the hero image on the landing page

The landing hero uses the first shoot's hero image. Reorder `shoots.js` (first entry becomes the hero), or change that shoot's `heroIndex` to pick a different frame from the set.

## Design tokens

Edit `tailwind.config.js`:

- `paper` (#F1EADE) — background
- `ink` (#2A231D) — primary text
- `muted` (#7A6E5F) — secondary text
- `rust` (#8C3F2E) — accent (asterisks, hovers)
- `hairline` (#D7CDBD) — rules and borders

Fonts (Cormorant Garamond + EB Garamond + Inter) are loaded from Google Fonts in `app/layout.jsx`.

## Grain overlay

The film grain is an inline SVG noise texture in `globals.css` (see `.grain-overlay`). To make it heavier, raise the `opacity` value (default `0.12`). To remove it, delete the `<GrainOverlay />` line in `app/layout.jsx`.
